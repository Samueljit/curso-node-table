## Como Ejecutar

```

Este programa calcula la tabla de multiplicar de la base especificada y te crea un archivo con el resultado en la carpeta raiz del proyecto.

```

```

* Asegurate de tener nodejs/npm instalando
* Instala las dependencias

$ npm install

* Ejecuta el ejemplo integrado

$ npm run base3

* Ejecuta el programa, ejemplo: 

$ node app.js -b 5 -l -h 15

Options:
      --help     Show help                                             [boolean]
      --version  Show version number                                   [boolean]
  -b, --base     Es la base de la tabla de multiplicar       [number] [required]
  -l, --listar   Muestra la tabla en consola          [boolean] [default: false]
  -h, --hasta    Es la cantidad hasta donde quieras la tabla
                                                          [number] [default: 10]
                                                          
```
